# For Children Website

A simple and interactive website designed for children.

## Getting Started

To view the website, simply open the `index.html` file in your web browser.

## File Structure

*   `index.html`: The main page of the website.
*   `style.css`: The main stylesheet for the website.
*   `Picture/`: This directory contains all the images used in the website.
*   `favicon.ico`: The icon for the website.
